# GeoWeather
Tool in python3 which allows to obtain several information on a city :
- Current temperature
- Wind speed
- Location on the map
- Air quality

### Execution
python3 geoweather.py
## Installation

Clone this repo
```bash
git clone https://github.com/snuv/geoweather.git
```


Then install all required components 

Required libs :
```
tk 
urllib 
numpy 
matplotlib 
basemap 
requests
PIL
```
    
## Deployment

To deploy this project run

```bash
python3 geoweather.py
```
